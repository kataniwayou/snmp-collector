8. `DeviceName_CaseInsensitive` — RecordFailure("DEVICE-A") then GetFailureCount("device-a") returns 1. Verifies StringComparer.OrdinalIgnoreCase on the dictionary.

```csharp
namespace SnmpCollector.Pipeline;

public sealed class SharpSnmpClient : ISnmpClient
{
    public Task<IList<Variable>> GetAsync(
        VersionCode version,
        IPEndPoint endpoint,
        OctetString community,
        IList<Variable> variables,
        CancellationToken ct)
        => Messenger.GetAsync(version, endpoint, community, variables, ct);
}
```

Update MetricPollJob to inject `ISnmpClient` instead of calling `Messenger.GetAsync` directly. Register `SharpSnmpClient` as singleton in `AddSnmpPipeline` (or `AddSnmpScheduling`).
