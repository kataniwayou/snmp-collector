**DeviceInfo.cs** -- Sealed record (NOT copied from Simetra -- different shape):
```csharp
public sealed record DeviceInfo(
    string Name,
    string IpAddress,
    string CommunityString,
    IReadOnlyList<MetricPollInfo> PollGroups);
```
- Name = human-readable device name (used as `agent` label value)
- IpAddress = string IP for display/logging
- CommunityString = resolved value (per-device override OR global default)
- PollGroups = runtime poll group list
- No DeviceType, no TrapDefinitions

**MetricPollInfo.cs** -- Sealed record (NEW, no Simetra equivalent):
```csharp
public sealed record MetricPollInfo(
    int PollIndex,
    IReadOnlyList<string> Oids,
    int IntervalSeconds)
{
    public string JobKey(string deviceName) => $"metric-poll-{deviceName}-{PollIndex}";
}
```
- PollIndex = zero-based position in device's MetricPolls list (DEVC-04)
- JobKey helper produces Quartz job identity: "metric-poll-{deviceName}-{pollIndex}"

**IDeviceRegistry.cs** -- Adapted from `src/Simetra/Pipeline/IDeviceRegistry.cs`:
- `bool TryGetDevice(IPAddress senderIp, [NotNullWhen(true)] out DeviceInfo? device)` -- trap path lookup
- `bool TryGetDeviceByName(string deviceName, [NotNullWhen(true)] out DeviceInfo? device)` -- poll path lookup
- `IReadOnlyList<DeviceInfo> AllDevices { get; }` -- for cardinality audit and Quartz job enumeration (NEW vs Simetra)
- Uses `System.Diagnostics.CodeAnalysis` for `[NotNullWhen]` and `System.Net` for `IPAddress`

**DeviceRegistry.cs** -- Adapted from `src/Simetra/Pipeline/DeviceRegistry.cs` with key changes:
- Use `FrozenDictionary<IPAddress, DeviceInfo>` for `_byIp` and `FrozenDictionary<string, DeviceInfo>` for `_byName` (NOT plain Dictionary as in Simetra)
- Constructor takes `IOptions<DevicesOptions> devicesOptions` and `IOptions<SnmpListenerOptions> snmpOptions`
- REMOVE all `IDeviceModule`/`IVirtualDeviceModule` logic (no module system)
- Community string resolution: `string.IsNullOrWhiteSpace(d.CommunityString) ? globalCommunity : d.CommunityString`
- Build `MetricPollInfo` list from `d.MetricPolls.Select((p, i) => new MetricPollInfo(i, p.Oids.AsReadOnly(), p.IntervalSeconds))`
- IP normalization: `IPAddress.Parse(d.IpAddress).MapToIPv4()` on both stored keys and in `TryGetDevice`
- Name lookup uses `StringComparer.OrdinalIgnoreCase`
- Store `_allDevices` as `IReadOnlyList<DeviceInfo>`
- Must `using System.Collections.Frozen;` for `ToFrozenDictionary()`

**IOidMapService.cs** -- NEW interface:
```csharp
public interface IOidMapService
{
    string Resolve(string oid);
    int EntryCount { get; }
}
```
- `Resolve` returns metric_name or "Unknown" (MAP-03)
- `EntryCount` used by cardinality audit

**OidMapService.cs** -- NEW class implementing IOidMapService and IDisposable:
- Constructor takes `IOptionsMonitor<OidMapOptions> monitor` and `ILogger<OidMapService> logger`
- Initialize `volatile FrozenDictionary<string, string> _map` from `monitor.CurrentValue.Entries.ToFrozenDictionary(StringComparer.Ordinal)`
- Store `_changeToken = monitor.OnChange(OnOidMapChanged)` -- the IDisposable returned by OnChange
- `Resolve(string oid)` returns `_map.TryGetValue(oid, out var name) ? name : Unknown`
- `public const string Unknown = "Unknown";` -- static constant for the fallback name
- `EntryCount => _map.Count`
- `OnOidMapChanged(OidMapOptions newOptions, string? name)`:
  - Capture old map, build new FrozenDictionary from newOptions.Entries
  - Compute diff: added = newMap.Keys.Except(oldMap.Keys).Count(), removed = oldMap.Keys.Except(newMap.Keys).Count(), changed = intersection where values differ
  - Atomic field swap: `_map = newMap`
  - Log at Information: "OID map reloaded: {Added} added, {Changed} changed, {Removed} removed. Total: {Total} entries"
- `Dispose()` calls `_changeToken?.Dispose()`
