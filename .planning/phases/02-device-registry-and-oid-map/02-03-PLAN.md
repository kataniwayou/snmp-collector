**TestOptionsMonitor helper:**
Create `tests/SnmpCollector.Tests/Helpers/TestOptionsMonitor.cs` -- a mutable `IOptionsMonitor<T>` implementation that allows tests to simulate OnChange callbacks:
```csharp
internal sealed class TestOptionsMonitor<T> : IOptionsMonitor<T>
{
    private Action<T, string?>? _listener;
    public TestOptionsMonitor(T initial) => CurrentValue = initial;
    public T CurrentValue { get; private set; }
    public T Get(string? name) => CurrentValue;
    public IDisposable? OnChange(Action<T, string?> listener)
    {
        _listener = listener;
        return null;
    }
    public void Change(T newValue)
    {
        CurrentValue = newValue;
        _listener?.Invoke(newValue, null);
    }
}
```
The `Change(T newValue)` method lets tests trigger the hot-reload callback with new options values.
