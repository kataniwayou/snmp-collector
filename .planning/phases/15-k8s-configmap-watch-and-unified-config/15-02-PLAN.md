  <files>src/SnmpCollector/Services/DynamicPollScheduler.cs</files>
  <action>
    Create `src/SnmpCollector/Services/DynamicPollScheduler.cs` with:

    ```
    namespace SnmpCollector.Services;
    ```

    Class: `public sealed class DynamicPollScheduler`
    Constructor dependencies:
    - `ISchedulerFactory schedulerFactory`
    - `IJobIntervalRegistry intervalRegistry` (cast to JobIntervalRegistry or use interface -- use interface with new Unregister)
    - `ILivenessVectorService livenessVector`
    - `ILogger<DynamicPollScheduler> logger`

    Main method: `public async Task ReconcileAsync(IReadOnlyList<DeviceOptions> newDevices, CancellationToken ct)`

    Implementation:
    1. Get the scheduler: `var scheduler = await _schedulerFactory.GetScheduler(ct);`
    2. Get current poll job keys: `var allKeys = await scheduler.GetJobKeys(GroupMatcher<JobKey>.GroupEquals("DEFAULT"), ct);`
       Filter to poll jobs: `var currentPollKeys = allKeys.Where(k => k.Name.StartsWith("metric-poll-")).ToDictionary(k => k.Name);`
    3. Build desired job set from newDevices:
       - For each device, for each MetricPoll (indexed), build key `$"metric-poll-{device.Name}-{pi}"`.
       - Store as `Dictionary<string, (string DeviceName, int PollIndex, int IntervalSeconds)>`.
    4. Compute diff:
       - `toRemove` = currentPollKeys.Keys.Except(desiredKeys)
       - `toAdd` = desiredKeys.Except(currentPollKeys.Keys)
       - `toCheck` = currentPollKeys.Keys.Intersect(desiredKeys) -- check if interval changed
    5. Remove stale jobs:
       - `await scheduler.DeleteJob(key, ct)` for each toRemove key.
       - `_intervalRegistry.Unregister(keyName)` for each removed.
       - `_livenessVector.Remove(keyName)` for each removed.
       - Log: "Removed poll job {JobKey}"
    6. Add new jobs:
       - Build `IJobDetail` with `JobBuilder.Create<MetricPollJob>()` using `.WithIdentity(jobKey)`, `.UsingJobData("deviceName", ...)`, `.UsingJobData("pollIndex", ...)`, `.UsingJobData("intervalSeconds", ...)`.
       - Build `ITrigger` with `TriggerBuilder.Create()`, `.WithIdentity($"{keyName}-trigger")`, `.StartNow()`, `.WithSimpleSchedule(s => s.WithIntervalInSeconds(interval).RepeatForever().WithMisfireHandlingInstructionNextWithRemainingCount())`.
       - `await scheduler.ScheduleJob(job, trigger, ct)`.
       - `_intervalRegistry.Register(keyName, interval)`.
       - Log: "Added poll job {JobKey} interval={IntervalSeconds}s"
    7. Check existing jobs for interval changes:
       - For each toCheck key, compare desired interval with `_intervalRegistry.TryGetInterval(keyName, out var current)`.
       - If interval changed: `await scheduler.RescheduleJob(triggerKey, newTrigger, ct)`, update interval registry.
       - Log: "Rescheduled poll job {JobKey} interval {Old}s -> {New}s"
    8. Log summary: "Poll job reconciliation complete: +{Added} added, -{Removed} removed, ~{Rescheduled} rescheduled, {Total} total"

    IMPORTANT: Import Quartz namespace. Use `Quartz.Impl.Matchers.GroupMatcher<JobKey>`. The `Jobs.MetricPollJob` type is needed for `JobBuilder.Create<MetricPollJob>()` -- add `using SnmpCollector.Jobs;`.
  </action>

    Class: `public sealed class ConfigMapWatcherService : BackgroundService`

    Constructor dependencies:
    - `IKubernetes kubeClient` (k8s.IKubernetes -- already registered as singleton in DI)
    - `IOidMapService oidMapService` (cast to OidMapService for UpdateMap, or use IOidMapService with UpdateMap on interface -- it was added in Plan 01)
    - `IDeviceRegistry deviceRegistry` (cast to DeviceRegistry for ReloadAsync, or use IDeviceRegistry with ReloadAsync on interface -- it was added in Plan 01)
    - `DynamicPollScheduler pollScheduler`
    - `ILogger<ConfigMapWatcherService> logger`

    Constants:
    - `private const string Namespace = "simetra";` (or read from env var / downward API, but hardcode for now matching existing K8sLeaseElection pattern)
    - `private const string ConfigMapName = "simetra-config";`
    - `private const string ConfigKey = "simetra-config.json";`
    - `private static readonly TimeSpan ReconnectDelay = TimeSpan.FromSeconds(5);`

    Fields:
    - `private readonly SemaphoreSlim _reloadLock = new(1, 1);`

    JSONC parsing options (static readonly):
    ```csharp
    private static readonly JsonSerializerOptions JsoncOptions = new()
    {
        ReadCommentHandling = JsonCommentHandling.Skip,
        AllowTrailingCommas = true,
        PropertyNameCaseInsensitive = true
    };
    ```

    Method: `protected override async Task ExecuteAsync(CancellationToken stoppingToken)`
    Implementation:
    1. Initial load: call `await LoadFromConfigMapAsync(stoppingToken)`.
    2. Watch loop (while !stoppingToken.IsCancellationRequested):
       ```csharp
       try
       {
           var response = _kubeClient.CoreV1.ListNamespacedConfigMapWithHttpMessagesAsync(
               Namespace,
               fieldSelector: $"metadata.name={ConfigMapName}",
               watch: true,
               cancellationToken: stoppingToken);

           await foreach (var (type, configMap) in response
               .WatchAsync<V1ConfigMap, V1ConfigMapList>(cancellationToken: stoppingToken))
           {
               if (type == WatchEventType.Modified)
               {
                   await HandleConfigMapChangedAsync(configMap, stoppingToken);
               }
           }
       }
       catch (OperationCanceledException) when (stoppingToken.IsCancellationRequested)
       {
           break;
       }
       catch (Exception ex)
       {
           _logger.LogWarning(ex, "ConfigMap watch disconnected, reconnecting in {Delay}s", ReconnectDelay.TotalSeconds);
           await Task.Delay(ReconnectDelay, stoppingToken);
       }
       ```

    Method: `private async Task LoadFromConfigMapAsync(CancellationToken ct)`
    - Read ConfigMap: `var cm = await _kubeClient.CoreV1.ReadNamespacedConfigMapAsync(ConfigMapName, Namespace, cancellationToken: ct);`
    - Extract key: `cm.Data[ConfigKey]`
    - Parse and apply via `ApplyConfigAsync`.
    - Log: "Initial config loaded from ConfigMap {ConfigMapName}/{ConfigKey}"

    Method: `private async Task HandleConfigMapChangedAsync(V1ConfigMap configMap, CancellationToken ct)`
    - Guard: `if (!configMap.Data.TryGetValue(ConfigKey, out var json)) return;`
    - Serialize reload: `await _reloadLock.WaitAsync(ct);` try/finally `_reloadLock.Release();`
    - Call `await ApplyConfigAsync(json, ct);`
    - Log: "ConfigMap change detected, config reloaded"

    Method: `private async Task ApplyConfigAsync(string json, CancellationToken ct)`
    - Parse: `var config = JsonSerializer.Deserialize<SimetraConfigModel>(json, JsoncOptions) ?? throw new InvalidOperationException("...");`
    - Update OID map: `_oidMapService.UpdateMap(config.OidMap);`
    - Reload devices: `await _deviceRegistry.ReloadAsync(config.Devices);`
    - Reconcile poll jobs: `await _pollScheduler.ReconcileAsync(config.Devices, ct);`
    - Log: "Config applied: {OidCount} OIDs, {DeviceCount} devices"

    IMPORTANT: The Namespace should be read from the pod's namespace file if available:
    ```csharp
    private static string GetNamespace()
    {
        const string nsFile = "/var/run/secrets/kubernetes.io/serviceaccount/namespace";
        return File.Exists(nsFile) ? File.ReadAllText(nsFile).Trim() : "simetra";
    }
    ```
    Call this in the constructor and store in a readonly field.

    Required usings: `k8s`, `k8s.Models`, `System.Text.Json`, `SnmpCollector.Configuration`, `SnmpCollector.Pipeline`.
